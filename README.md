# Mensatt Backend

The backend repository for Mensatt written in go.

## Dev Server with hot reload

Prequesites: `docker`  

Create and start: `make up`  
Stop and remove: `make down`  
Start: `make stop`  
Stop: `make stop`

## Dev Tools

Generate all: `make generate`
Run gqlgen: `make gqlgen`  
Run sqlc: `make sqlc`
